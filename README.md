# TypeScript To-Do App

A simple **To-Do application** built with **TypeScript** and **Node.js**, running in the terminal.  
This project is designed to help beginners learn TypeScript while building a practical application.

---

## Features

- Add tasks to a to-do list
- List all tasks with status (Pending / Done)
- Mark tasks as completed
- Delete tasks
- Interactive terminal menu
- Fully typed with TypeScript
